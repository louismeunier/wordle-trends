# Wordle Trends

A minimal app to explore the effect of wordle on Google search trends.

Built with R + Shiny.

## Development

The app logic lives at [app.R](app.R). The easiest way to set-up all the dependencies is to run:

```sh
    ./bin/dependencies.sh
```

You can then run

```sh
    ./bin/serve.sh
```

to start the development server.
